从零开始Gridea+Github搭建个人博客

2021-06-22 2 min read # 笔记 # Gridea

👩‍💻要科学上网


概述

Gridea
一个静态博客写作客户端，简单来说，就是不用进代码页面进行HTML设计，直接使用Markdown语法进行写作的软件。
选择适合自己的电脑版本下载安装即可。

Github
👩‍💻要科学上网，或者运气好可以进去
一个面向开源及私有软件项目的托管平台。Github负责提供站点，在Gridea中完成的博客同步到Github进行代码托管后，你可以通过域名在浏览器中查看自己的博客网站。
若没有账号，需前往注册账号。


Github注册

  1. 进入Github首页通过邮件注册账号
  2. Choose Free
  3. Skip this step跳过个性化设置（也可以自己随意）
  4. 验证邮箱
  5. 登录成功


Github创建仓库

  1. 点击Start a Project按钮创建项目
    2.进行如下操作


Github创建令牌

  1. 点击右上角的头像，在菜单中选择Setting（设置）
  2. 将页面下滑，在左侧列表中选择Developer settings（开发者设置）
  3. 左侧列表中选择Personal access tokens（个人访问令牌）
  4. 按图示操作
  5. 下滑单击Generate token（生成令牌）按钮
  6. 保存令牌，令牌序列只会出现一次，注意复制保存（若丢失可以重新创建令牌）



配置Gridea

  1. 在Gridea官网下载适合自己电脑的版本
  2. 安装成功后启动Gridea（白屏正常现象，耐心等待）
  3. 出现界面后，在左侧列表点击远程
  4. 按图示操作
  5. 配置完成，可以开始写博客了，通过预览可以查看当前博客的界面效果，左下角小地球图标可以进入自己的实际博客网站查看界面表现。


遇到的问题

  1. 远程连接失败？
    答：查看是否科学上网。或者配置出错，可在搜索引擎中查找解答。

  2. 同步失败？
    答：可以点击蓝色部分查找解答。

    或者点击【编辑】，选择【开发者工具】，查看具体原因。
